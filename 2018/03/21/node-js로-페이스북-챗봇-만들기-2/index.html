1. 개발을 하기 위한 환경 준비

* node.js

먼저 node.js를 설치하세요. node.js 공식 홈페이지

버전이 두 개예요.
8.10.0 LTS
9.8.0 현재버전

아마 미래에 이 글을 보게될 경우 버전은 당연히 사진과 다릅니다. 아무 버전이나 깔아도 상관 없어요. 정 못 고르시겠으면 동전을 던지고 앞면이 나오면 오른쪽 뒷면이 나오면 왼쪽 버전을 선택하세요.

설치하는 과정은 간단합니다. 다운로드 받은 파일을 실행하고 다음 다음 다음 다음을 눌러주세요. 끈기를 잃지 않고 다음을 반복해서 누르시면 설치가 완료됩니다.

단, 이후 설치가 되었는지 확인하는 과정이 반드시 필요합니다. 터미널에서 다음 명령어를 입력하세요.

사진으로 보여드릴게요.

터미널이 뭔지 모르시는 분에게
터미널은 윈도우를 쓰신다면 왼쪽 하단의 윈도우 버튼을 누르신 후 “명령 프롬프트”라고 입력하시면 나오는 애예요. 걔를 실행시키고 “ipconfig”이라고 입력해보세요. 그럼 ip를 알 수 있어요.(windows의 경우)

명령 프롬프트라고 해도 안 나오면

누르신 다음에

를 입력하고 확인을 누르세요.
참고로 윈도우 버튼은 키보드 좌측 Ctrl 키와 Alt 키 사이에 있어요. (Mac을 쓰신다면 달라요! 물어보지 마세요. 인터넷에 검색해요!)

글이 불필요하게 길어지고 있네요.

무튼 node.js를 깔았으면 터미널에서 버전을 확인하는 명령어를 입력하고 제대로 설치가 되었는지 확인하세요. 반드시.

* Visual Studio Code

“비쥬얼 스튜디오 코드”라고 읽으시면 돼요. 설치하러 가 봐요. Visual Studio Code 공식 홈페이지

아 뭔지 설명해야겠구나. 얘는 개발할 때 쓰는 메모장 상위호환 판이에요. 좋은 아이죠. 가벼운 웹 개발을 하시는 분들은 고급 메모장을 많이 사용하시는데요. 생각나는 애들이 Sublime Text, Atom 등이 있어요. 물론 메모장..이 아니라 텍스트 에디터라고 할게요. 텍스트 에디터는 참 많아요. 근데 javascript 개발할 땐 제 생각에 VSCode(Visual Studio Code)가 짱이에요. 그렇다고요. 그러니까 VSCode를 깔아요.

위에 보시면 Download for Windows가 있어요. 맥 쓰시면 맥 버전용, Linux 쓰시면 해당 버전에 맞는 다운로드 버튼이 나타날 겁니다. 그냥 다운 받으세요. 그리고 깔아요. 설치 과정은 node.js와 동일해요. 그냥 다음을 계속 눌러요.

여기 멋진 분이 쓴 글이 있네요. 설치 과정을 잘 적어두셨어요. 참고해요.

한 가지 Tip을 드리자면 설치 도중 “Code(으)로 열기”작업을 Windows 탐색기 파일의 상황에 맞는 메뉴에 추가, 디렉터리의 상황에 맞는 메뉴에 추가 항목을 체크하세요. 폴더에서 코드 열 때 편해요!

폴더나 소스 파일 마우스 우클릭 시 중간에 Open with Code라고 생겨요.
